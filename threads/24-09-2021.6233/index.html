● Main
Cheat has been updated for the latest game version
Now our product supports disabling lag compensation, using the command "cl_lagcompensation 0". This will allow you to shoot faster at opponents who use the lag/defensive exploit.
Note: this command can be changed only when you are in the game menu, or already on the map in the spectators.
Warning: disabling lag compensation may affect the accuracy of the aimbot and its operation in general, use it only when necessary.

● Misc
Added smoothing.

P. S. You need to re-download the loader, who will have problems with the previous one. Also, 1 additional day was added to all users with an active subscription.
